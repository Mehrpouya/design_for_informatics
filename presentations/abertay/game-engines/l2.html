##Landscape - Create a new project - Create an empty level (Ctrl + N || Menu=>file=>new level) - Add an atmospheric fog, so it is not pitch black!
##Spend some time playing with Sculpt tools
##Important interactions - Ctrl - Left mouse button: **heightens || increases** - Shift + Left mouse button: **lowers || decreases** - Ctrl + Z: **undo** - Ctrl + Y: **redo**
###Painting your landscape - Download landscape material: - [compressed as 7z](/presentations/res/game-engines/l2Res/content/landscapeMaterial.7z) - [compressed as tar](/presentations/res/game-engines/l2Res/content/landscapeMaterial.tar) - [compressed as zip](/presentations/res/game-engines/l2Res/content/landscapeMaterial.zip) - Extract/decompress the downloaded file - Copy the contents inside your unreal project folder - Fix the my_landscape_material links to the textures. You do this, by double clicking on the landscape_material and we will cover the process in the class.
