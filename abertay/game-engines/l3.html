###Texture formats - bmp - float - pcx - png - psd - tga - jpg - exr - dds - Cubemap Texture (32bits/Channel, 8.8.8.8 ARGB 32 bpp, unsigned) See Cubemap Tools for more information. - hdr - Cubemap Texture (LongLat unwrap)
- Expressions are snippets of HLSL code (High Level Shading Language) - Remember values are normally between 0-1 in UE4 - Many things are stored inside RGBA values, so when you use textures, have a look at those values! - Refraction: bent of light ray when it enters different substances - Material instance: UE4 feature, so you can save processing two materials by making instances of one material and changing top level parameters.
